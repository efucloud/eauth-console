# EAuth (Frontend)
EAuth is an enterprise-grade authentication platform that provides multiple authentication methods, including Face Recognition, OpenID Connect (OIDC), and Web Authentication. It supports Multi-Factor Authentication (MFA) and flexible token generation settings, enabling unified account access across different applications. It helps enterprises achieve fine-grained authentication management, complete authentication flows quickly, and improve overall authentication efficiency.

For backend and deployment files, please refer to: https://github.com/efucloud/eauth
